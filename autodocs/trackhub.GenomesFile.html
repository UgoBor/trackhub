trackhub.GenomesFile¶

class trackhub.GenomesFile(genome=None)[source]¶

Bases: trackhub.base.HubComponent

Represents the genomes file on disk. Can contain multiple Genome objects, each of which represent a stanza in this file.

The file ultimately created (with the self.render() method) will be determined by the parent Hub’s genome_filename attribute. By default, this is the hub name, plus ”.genomes.txt”
