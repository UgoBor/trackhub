trackhub.Genome¶

class trackhub.Genome(genome, trackdb=None, genome_file_obj=None)[source]¶

Bases: trackhub.base.HubComponent

Represents a 2-line genome stanza within a “genomes.txt” file.

The file itself is represented by a GenomesFile object.
